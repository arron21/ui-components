Welcome to Blonkz UI

This is a demo of the UI components

Getting Started

To get started, you can run the following command to run the installation script.

npx @blonkz@latest init
Importing Components

After installing the package, you can import the components you need into your project.

npx @blonkz@latest add card
Styling the components

The components are styled using CSS variables. You can override the variables in the ui/css/_theme.scss file.

You may also modify and of the other CSS files to better suit your needs.
